# Project Brief Website Aruna Karsa

## User Story
Sebagai user, saya ingin mengenalkan perusahaan Aruna Karsa kepada publik melalui website. Website ini akan menjadi alat bantu untuk pihak marketing dalam mencari klien.

## Requirements
- UI Website minimalis, elegan, modern, profesional.
- Tampilan website harus responsive di berbagai perangkat.
- Website harus memiliki halaman Home, About, Services, Portfolio, Contact, dan Blog.

## UI Page
- Header
- Footer
- Home Page
- About Page
- Services Page
- Portfolio Page
- Contact Page
- Blog Page

### Header
Header website Aruna Karsa akan menampilkan logo Aruna Karsa dan navigasi ke halaman lain. Berikut beberapa hal yang harus ada di header:
- Logo Aruna Karsa
- Navigasi ke halaman lain

### Footer
Footer website Aruna Karsa akan menampilkan informasi tentang perusahaan, layanan, proyek, dan kontak. Berikut beberapa hal yang harus ada di footer:
- Logo Aruna Karsa
- Informasi perusahaan
- Informasi kontak
- Link ke media sosial

### Home Page
Halaman utama website Aruna Karsa. Halaman ini akan menampilkan informasi tentang perusahaan, layanan, proyek, dan kontak. Berikut beberapa hal yang harus ada di halaman Home: 
- Header dengan logo Aruna Karsa dan navigasi ke halaman lain. 
- Hero section dengan tagline Aruna Karsa dan tombol untuk menghubungi kami. 
- About section dengan informasi singkat tentang perusahaan. 
- Services section dengan informasi tentang layanan yang ditawarkan. 
- Portfolio section dengan menampilkan beberapa proyek yang telah selesai. 
- Contact section dengan formulir kontak dan informasi kontak. 
- Footer dengan logo Aruna Karsa dan informasi kontak.
